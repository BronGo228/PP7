Тема практики Соадминистрирование баз данных и серверов

Приложение обнаружения дефектов экранов смартфонов  
Язык: **Python**  
GUI: **Tkinter**  
БД: **Microsoft SQL Server**  
IDE: **Visual Studio Code**  

Функции:
- Подключение к MS SQL Server через pyodbc  
- CRUD для таблиц Устройства, Проверка, Дефекты, Пользователи  
- Администрирование ролей и пользователей SQL Server  
- Диалоговые окна для добавления/изменения записей  

Содержимое:
- main.py: основное приложение с GUI, подключение, CRUD, админ-функции, вызов формы настроек
- dialogs.py: модальные диалоги для добавления/редактирования
- connection_dialog.py — новая форма настройки подключения
- helpers.py: подключение к СУБД
- db_schema.sql: скрипт создания БД и таблиц
- config.json: Конфигурация
- requirements.txt — зависимости
